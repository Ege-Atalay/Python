# Formatlama
```sh
"{} {} {} {}".format(45,365,59.78,47)
```
'45 365 59.78 47'
```sh
a = 6
b = 7
print ("{} * {} 'nin çarpımından çıkan sonuç {} 'dir. ".format(a,b,a*b))
```
6 * 7 'nin çarpımından çıkan sonuç 42 'tur. 
```sh
"{: .3f} {: .2f} {: .4f} ".format(45.45652, 85.57225, 74.545355)
```
' 45.457  85.57  74.5454 '
