# if, else ve elif
## if
**if komutu** bir koşul belirler ve bu koşulauyulup uyulmama durumunda kodu dallandırır.

```sh
a = 3
b = 5

if (a < b):
  print("Merhaba Ege")
```
Merhaba Ege

## else
**else komutu** if komutundaki ifade yanlış ise son ihtimal olarak çalışması istenen koşul komutudur.
```sh
a = 9
b = 5

if (a < b):
    print("Merhaba Ege")
else:
    print("Ege Atalay")
```
Ege Atalay
